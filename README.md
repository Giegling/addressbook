# Installation #

1.Make a clone of this repository - https://github.com/brulla/addressbook.git

2.Install node modules
```
#!bash

cd addressbook
npm install
```
3.Install bower components
```
#!bash

cd client/assets
bower install
```
4.Generate javascript sources

```
#!bash

gulp
```

5.Run server
```
#!bash

cd ../..
npm start
```
